> “Start simple. Build consistently.”

这是我博客的第一篇文章。

折腾了半天，终于把这个 blog 跑起来了。过程其实不复杂，但细节挺多。

一开始只是想找个地方记录学习内容，尤其是最近在学校学的一些东西：

- Search（A*, Greedy）
- Planning（PDDL）
- Heuristics（h*, h+）
- MDP / Reinforcement Learning  

与其把这些东西散在笔记里，不如整理成 blog，一边写一边加深理解。

## 踩过的坑

简单记录几个刚踩过的：

- 差点又买了个binarylane伺服器，才发现Github Page这个神仙工具
- **baseurl 配错 → CSS 全 404**
- URL 拼接多一个空格 → `%20` bug
- repo 不是 `username.github.io` → 路径问题
- canonical / redirect 搞乱访问路径
这些问题都不难，但确实容易卡住。

## 接下来会写什么？

这个 blog 主要会记录：

- AI / ML 学习笔记
- 一些算法理解（尽量写清楚“为什么”）
- 项目 / 实践过程
- 偶尔写点技术之外的东西

---

如果你刚好也在学类似的内容，希望这些记录能帮到你一点。
